## 🌟 About The Project

This is **Ziad Mohamed Al-Sayed’s developer portfolio**, designed to present professional experience, projects, skills, and contact links in an elegant and responsive web interface.

The portfolio highlights recent full-stack work including React apps, API integrations, and real live deployed projects. It serves as a central hub for recruiters, collaborators, and clients to explore Ziad’s expertise and get in touch.

Built using **React.js** and styled with **Tailwind CSS**, the portfolio is fully responsive and optimized for modern web performance.

📦 Getting Started (Local Development)
🧾 Prerequisites

You’ll need:

Node.js installed (https://nodejs.org/)

npm or Yarn

📥 Installation

1- Clone the repository:

git clone https://github.com/ziad-mohamed-mern/InternIntelligence_Portfolio.git

2- Navigate into the project directory:

cd InternIntelligence_Portfolio

3- Install the dependencies:

npm install

4- Start the development server:

npm start

Your portfolio will run at:

http://localhost:3000

🌍 Deployment

This portfolio is deployed using Vercel:

🔗 https://ziad-mohamed-alsayed.vercel.app/

Vercel automatically manages builds and deployment on each push to the main branch.
